# Sin Land

> OpenSiv3D で制作した2Dアクション謎解きゲームです

---

## ⚙️ インストール方法

### 🪟 Windows

# ZIPをダウンロードする場合
[⬇️ SinLand_Windows.zip をダウンロード](SinLand.zip)
1. ZIPファイルを解凍してください。
2. 解凍されたフォルダに入っているSinLand.exeを実行してください。
3. もしSinLand.exeを移動したい場合は、SinLand.exeと同じディレクトリ内にAssetsフォルダを一緒に配置してください。音声などが正常に再生されなくなります。

# git cloneする場合 
1. このリポジトリをクローンしてください。
   ```bash
   git clone https://github.com/haitooo/SinLand.git
2. クローンしたリポジトリからSinLand.exeを実行してください。
3. もしSinLand.exeを移動したい場合は、SinLand.exeと同じディレクトリ内にAssetsフォルダを一緒に配置してください。音声などが正常に再生されなくなります。

### 🍎🐧 Apple・Linux
1. [公式サイト](https://siv3d.github.io/ja-jp/) から **OpenSiv3D (v0.6 以降)** をインストール
2. Main.cppをこのリポジトリのMain.cppに差し替えてください。
3. ビルドして実行ファイルを作成してください。
4. 実行ファイルと同じディレクトリ内にAssetsフォルダを一緒に配置してください。音声などが正常に再生されなくなります。
5. 実行ファイルを実行でプレイできます。

## 🎮 ゲーム概要

プレイヤーは“罪の街”に迷い込んだ人物として、各ステージに隠されたルールを見抜きながら出口を目指します。  
各ステージはそれぞれ独立したテーマとギミックを持ち、プレイヤーの直感と観察力が試されます。

### 🕹 操作方法

| 操作 | 内容 |
|------|------|
| ← / → または **A / D** | 左右移動 |
| ↑ または **W / Space** | ジャンプ |
| **Esc** | タイトルに戻る（タイトル画面でEsc：アプリを終了します） |

## 🔍 各ステージ解説

### Stage 1 森林  
> BGMが途絶えた街で、わずかな環境音がヒントになる。  
> プレイヤーの動きを通して「音」を感じさせる演出を意識。

---

### Stage 2 鏡の部屋
> **左右対称の心理トリック**。  
> プレイヤーと“鏡像のプレイヤー”が同時に動く仕掛け。  
> 「自分の操作＝もう一人を制御する」ことへの違和感を演出。

---

### Stage 3 折れた橋
> **物理と感情のリンク**。  
> シャーペンのような橋が折れることで道が変化。  
> 折れた瞬間の震えや音、そして落下する描写にこだわり。

---

### Stage 4 信号の交差点
> **社会的ルールを可視化**。  
> 赤信号では動くと“罰”が発生。  
> センサー範囲を極限まで狭め、緊張感のある動作を設計。

---

### Stage 5 真珠スロット
> **運と因果のステージ**。  
> スロットを止めて鍵とドアを揃える。  
> 真珠貝が開く瞬間のアニメーションと効果音に注力。

---

### Stage 6 分身の術
> **欺きと洞察**。  
> 忍者が近づくと四人に分身。  
> ヒントは「ノイズ」「影」「間」など微細な違い。  
> 三回正解でクリア、ミスでやり直しという緊張感を演出。
